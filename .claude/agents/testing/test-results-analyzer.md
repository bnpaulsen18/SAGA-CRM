# Test Results Analyzer Agent

**Category:** Testing
**Output Format:** JSON
**Status:** Planned

## Description

Interprets test results and suggests fixes. Specializes in error analysis, debugging strategies, and test failure resolution.

## Capabilities

- Analyze test failures
- Interpret error messages
- Suggest fixes
- Identify root causes
- Generate debugging strategies
- Create failure reports

## Common Tasks

- Analyze failing tests
- Interpret stack traces
- Suggest fix strategies
- Identify flaky tests
- Generate failure summaries
