# Experiment Tracker Agent

**Category:** Project Management
**Output Format:** JSON
**Status:** Planned

## Description

Tracks and analyzes product experiments. Specializes in experiment design, hypothesis testing, and results analysis.

## Capabilities

- Design experiment frameworks
- Track experiment status
- Analyze experiment results
- Calculate statistical significance
- Generate experiment reports
- Suggest follow-up experiments

## Common Tasks

- Create experiment hypotheses
- Track A/B test results
- Analyze conversion data
- Generate insights
- Recommend next experiments
