# cofounder/ — your AI cofounder's brain

This folder is the shared memory for your SAGA cofounder. In Claude Code, invoke it with **`/cofounder`** (default mode: fundraise):

```
/cofounder fundraise prep my first investor meeting
/cofounder strategize what should I focus on this week
/cofounder operate let's tighten the donor onboarding flow
```

## Files
- **company.md** — what SAGA is: product, business model, team, real status.
- **raise.md** — the $5M pre-seed plan + an honest reality check.
- **investors.md** — target investor profile + live pipeline.
- **pitch.md** — the narrative spine + answers to the tough questions.
- **tasks.md** — this week's focus + open commitments.

## The one rule that matters most
The SAGA product demos use **illustrative data** for a fictional customer ("Hope Foundation" — $11,840 MRR, 318 sustainers, ~41% retention, and so on). **Those are not SAGA's company metrics.** Never let them reach a deck, an email, or an investor. Real traction goes in `company.md`, clearly marked as real. Everything tagged `<!-- FILL IN -->` is unknown until you fill it.

## Keep it current
After real work, update these files — the cofounder will offer to. A stale brain gives confident, wrong advice.
